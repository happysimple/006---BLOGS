07. tabbed

$ brew install git

打开 https://git-scm.com/downloads，下载并安装 Git for Windows

```{eval-rst}
.. tabbed:: macOS

    ::
    
        $ brew install git

.. tabbed:: Windows

   打开 https://git-scm.com/downloads\ ，下载并安装 Git for Windows
```
